Welcome to BlockMark

The simple code editor for all skill levels

BlockMark is a tool that allows you to save frequently used code snippets for quick access. You can save blocks of code as buttons that you can insert into any text field with a single click. This can be useful for things like programming languages, website development, or any other task where you need to frequently use the same snippets. It can also help with moving from block based to text based coding.
